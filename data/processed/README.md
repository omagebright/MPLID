# Data Files

Data files are hosted on Zenodo due to size constraints.
Download from: https://doi.org/10.5281/zenodo.18487584

After downloading, extract the files to this directory.

## Files

| File | Description | Size |
|------|-------------|------|
| `train_residues.csv.gz` | Training split (2,578 proteins, 4,907,696 residues) | ~68 MB |
| `val_residues.csv.gz` | Validation split (1,051 proteins, 1,403,838 residues) | ~20 MB |
| `test_residues.csv.gz` | Test split (1,075 proteins, 1,743,791 residues) | ~24 MB |
| `protein_metadata.csv` | Per-protein summary statistics | ~217 KB |

## Dataset Summary

- **Total proteins**: 4,704
- **Total residues**: 8,055,325
- **Contact residues**: 80,439 (1.00%)
- **Sequence clusters**: 813 (30% identity)
- **Contact definition**: All heavy atoms within 4.0 A of lipid heavy atoms
